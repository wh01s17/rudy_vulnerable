R.U.D.Y. Vulnerable Django Application

This application is intentionally vulnerable to R.U.D.Y. attacks for educational purposes.

Vulnerable Endpoints:

  • Form Submission Endpoint - Vulnerable to slow POST attacks
  • File Upload Endpoint - Vulnerable to large file uploads

About R.U.D.Y. Attacks:

R.U.D.Y. (R-U-Dead-Yet) is a DoS attack that works by opening connections to a targeted server and then keeping those connections open for as long as possible by slowly sending data through them. This causes server resources to be consumed and eventually exhausts all connection pools, making the server unavailable to legitimate users.

Educational Purposes Only:

This application is meant for educational demonstrations of how R.U.D.Y. attacks work and how to defend against them. Please only use in controlled environments with proper authorization.
